# 💫 About Me:
I’m a Computer Science student growing into a Full-Stack and AI developer. I enjoy building systems that combine backend engineering with intelligent automation.<br><br>My work focuses on developing applications using Java, Spring Boot, SQL, and machine learning tools. I’m particularly interested in designing practical systems where traditional software and AI work together to solve real problems.<br><br>Some of my projects include an AI-enabled portfolio platform with a chatbot that retrieves information from a backend database to answer questions about my work, as well as automation tools like a Telegram spam filter.<br><br>I’m currently exploring AI-assisted applications, backend architecture, and automation systems.
